# PyParticle Visualization Package

## Overview

The PyParticle Visualization Package (`PyParticle.viz`) is designed to provide flexible and powerful tools for visualizing particle population data. This package is part of the larger PyParticle project, which focuses on building and managing particle populations in various scenarios.

## Features

- **Flexible Layouts**: Create customizable grid layouts for visualizing multiple scenarios or variables.
- **Dynamic Plotting**: Generate various types of plots, including line plots, to represent particle population data effectively.
- **Styling Options**: Apply different styles to plots, including colors and line styles, to enhance visual appeal and clarity.
- **Formatting Utilities**: Format axes, add legends, and customize plot elements for better presentation.

## Installation

To install the required dependencies, use the following command:

```
pip install -r requirements.txt
```

## Usage

To use the PyParticle Visualization Package, you can refer to the example scripts provided in the `examples` directory. The `partmc_demo.py` script demonstrates how to build a particle population and visualize it using the available plotting functions.
